# 术语表 · mail-ai

| 术语 | 定义 |
|------|------|
| IMAP | Internet Message Access Protocol，用于从邮件服务器同步邮件的协议 |
| SMTP | Simple Mail Transfer Protocol，用于发送邮件的协议 |
| 应用专用密码 | 第三方邮件服务提供的独立密码，用于授权非官方客户端访问邮箱 |
| 智能标签 | 由 AI 自动为邮件分配的分类标识，如"需要回复"、"待审批"等 |
| 优先级排序 | AI 根据邮件内容判断重要程度，将高优先级邮件置顶 |
| 待办项 | 从邮件正文中自动识别出的需要用户执行的动作 |
| 会话线程 | 同一主题下多封往来邮件组成的对话链 |
