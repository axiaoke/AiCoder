# 项目约束 · mail-ai

> TRD v1.0 确认后填入（2026-04-04）

| 约束项 | 内容 |
|--------|------|
| 邮件协议 | IMAP（收）/ SMTP（发），MVP 不支持 OAuth，使用应用专用密码 |
| 邮件存储 | 全量镜像到 MySQL；附件存本地磁盘，路径记字段，预留 MinIO 扩展 |
| 首次同步深度 | 最近 30 天邮件 |
| AI 服务 | 第三方 API（Claude 主 / GPT 备），单次调用完成分类+摘要+提取 |
| AI 调用并发 | Bull 队列，并发数 3，防触发 API 限流 |
| 飞书集成 | 必选，飞书 OAuth 为唯一登录方式；飞书任务同步需额外 OAuth 授权 |
| 用户规模 | ≤ 50 人，单服务器架构，不做分布式 |
| 移动端 | 飞书 H5 容器（webview），响应式布局 |
| 安全 | 内部部署，暂不做端到端加密；IMAP 密码与飞书 token AES-256 加密存储 |
| 附件大小 | 单个 ≤ 10MB |
